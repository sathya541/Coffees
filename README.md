# Coffees
Android Sample app implementingMMVM - clean architecture using Jetpack Compose

# Main Features
1. Kotlin
2. MVVM
3. Jetpack Compose
4. MutableState
	- Simpler than LiveData or Flow with respect to composables.
5. Hilt
6. Navigation Components
7. Retrofit2
8. Single Activity Architecture
9. App Themes
10. Light and Dark theme
	- Toggle between themes
